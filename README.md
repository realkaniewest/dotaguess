A daily Dota 2 hero guessing game inspired by Wordle and Dotadle.

## Features

- **8 Game Modes**: Classic, Silhouette, Ability, Ultimate, Stats, Emoji, Lore, Items
- **Daily Challenge**: Same hero for all players each day
- **Hint System**: Green/Yellow/Gray color feedback
- **Share Results**: Wordle-style emoji share
- **127 Heroes**: Full Dota 2 hero roster from OpenDota API
- **Dark Fantasy UI**: Gold accents, glow effects, animated transitions
- **Stats Tracking**: Win streaks, guess distribution, localStorage save

## Quick Start

### Prerequisites
- Node.js 18+
- npm

### Setup (first time only)

```bash
cd dotaguess

# Install backend deps
cd backend && npm install && cd ..

# Download hero data (requires internet)
node scripts/downloadHeroes.js

# Install frontend deps
cd frontend && npm install && cd ..
```

### Run

**Windows:** Double-click `start.bat`

**Manual:**
```bash
# Terminal 1 - Backend
cd backend
npm start

# Terminal 2 - Frontend
cd frontend
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Set **Root Directory** to `frontend`
4. Add environment variable: `NEXT_PUBLIC_API_URL=https://your-backend.railway.app`
5. Deploy backend to Railway or Render
